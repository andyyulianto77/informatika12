Indentasi otomatis: VSCode secara otomatis mengatur indentasi kode.

Visual Studio Code (VSCode) adalah alat yang sangat berguna untuk mengedit kode. Mari kita lihat beberapa fitur kerennya.

Misalnya, ketika kita membuat kode PHP baru, VSCode secara otomatis akan membantu kita menjaga agar kode kita tertata rapi dengan fitur indentasi otomatis. Ini sangat membantu, terutama saat kita bekerja dengan struktur kode yang kompleks seperti if dan loop.

  • <?php
  • // Get input.
  • $hewan = $_GET['hewan'];
  • $perluKacang = '';
  • if ($hewan == 'tupai') {
  • $perluKacang = 'Ambillah beberapa kacang tanah..';

Hal pertama yang Anda perhatikan, VSCode telah membuat beberapa indentasi:

Indentasi tidak bersifat opsional dalam kursus ini. Anda harus membuat indentasi kode dalam ifs dan loop, serta definisi fungsi (lebih lanjut tentang itu nanti). 

Periksa tab tersebut. Nama berkas berwarna merah. Jika Anda melihatnya, berarti VSCode telah melihat sesuatu yang dianggap sebagai kesalahan. Jangan abaikan laporan ini. VSCode terkadang salah, tetapi tidak sering.

Pelengkapan kode: VSCode menyarankan kata kunci, variabel, dan fungsi yang relevan saat kita mengetik, mempercepat proses penulisan kode.

Selain itu, VSCode juga memiliki fitur pelengkapan kode yang sangat pintar. Ia bisa menebak apa yang ingin kita tulis selanjutnya berdasarkan konteks kode yang ada. Ini sangat menghemat waktu kita, terutama saat kita bekerja dengan kode yang panjang dan kompleks.
