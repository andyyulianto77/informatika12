Menghubungkan lembar gaya 

Mari tambahkan garis pada kepala.​

  1. <html lang="id">
  2. <head>
  3. <meta charset="UTF-8">
  4. <title>Kucing Terlucu</title>
  5. <meta name="viewport" content="width=device-width, initial-scale=1, shrink-to-fit=no">
  6. <link rel="stylesheet" href="styles.css">
  7. </head>

Lihat baris ke 6:

<link rel="stylesheet" href="styles.css">

Itu ada di bagian head , jadi itu metadata. Ia memberi tahu browser untuk melihat ke dalam file styles.css, untuk mengetahui cara memberi gaya pada HTML. Kami akan menyimpan gaya secara terpisah dari HTML, sehingga Anda dapat menggunakan kembali gaya dengan mudah. 

styles.css terdiri dari aturan CSS, yang memberi tahu browser tentang font, warna, dan hal lainnya. 

Aturan CSS 

Inilah yang ada di styles.css.

Hal pertama yang perlu diperhatikan adalah bahasa ini tidak terlihat seperti HTML. Itu karena bahasa ini sepenuhnya berbeda, dengan formatnya sendiri. 

Ada banyak aturan . Semuanya berbentuk:

  • pemilih {
  • nama properti : nilai
  • ....
  • }

Pemilih memberi tahu browser aturan apa yang berlaku. Dalam kasus ini: 

  • body {
  • Cara menampilkan isi tag body.
  • }
  • h1 {
  • Cara menampilkan konten semua tag h1.
  • }
  • h2 {
  • Cara menampilkan konten semua tag h2.
  • }

Bentuk umum lagi:

  • pemilih{
  • nama properti : nilai;
  • ....
  • }

Nama properti adalah hal yang akan diubah pada elemen yang dipilih. Kami menggunakan tiga properti berbeda di sini: 

  • font-family:jenis huruf yang digunakan
  • background-color:warna latar belakang
  • color:warna teks

Bentuk umum lagi:

  • pemilih{
  • nama properti : nilai;
  • ....
  • }

Nilai adalah apa yang harus digunakan. Contoh:

  • sans-serif: Font sederhana tanpa dekorasi apa pun
  • cursive: Font yang terlihat seperti seseorang yang menulisnya, semacam itu
  • antiquewhite: Warna kuning kecoklatan muda
  • darkgreen: hijau tua
  • darkred: merah tua

Satukan semuanya.
