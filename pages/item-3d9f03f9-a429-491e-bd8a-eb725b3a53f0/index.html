Mari tambahkan garis pada kepala.​

  1. <html lang="id">

  4. <title>Kucing Terlucu</title>

Lihat baris ke 6:

<link rel="stylesheet" href="styles.css">

Itu ada di bagian head , jadi itu metadata. Ia memberi tahu browser untuk melihat ke dalam file styles.css, untuk mengetahui cara memberi gaya pada HTML. Kami akan menyimpan gaya secara terpisah dari HTML, sehingga Anda dapat menggunakan kembali gaya dengan mudah. 

styles.css terdiri dari aturan CSS, yang memberi tahu browser tentang font, warna, dan hal lainnya. 

Aturan CSS 

Inilah yang ada di styles.css.

Hal pertama yang perlu diperhatikan adalah bahasa ini tidak terlihat seperti HTML. Itu karena bahasa ini sepenuhnya berbeda, dengan formatnya sendiri. 

Ada banyak aturan . Semuanya berbentuk:

  • pemilih {
  • nama properti : nilai
  • ....
  • }

Pemilih memberi tahu browser aturan apa yang berlaku. Dalam kasus ini: 

  • body {
  • Cara menampilkan isi tag body.
  • }
  • h1 {
  • Cara menampilkan konten semua tag h1.
  • }
  • h2 {
  • Cara menampilkan konten semua tag h2.
  • }

Bentuk umum lagi:

  • pemilih{
  • nama properti : nilai;
  • ....
  • }

Nama properti adalah hal yang akan diubah pada elemen yang dipilih. Kami menggunakan tiga properti berbeda di sini: 

  • font-family:jenis huruf yang digunakan
  • background-color:warna latar belakang
  • color:warna teks

Bentuk umum lagi:

  • pemilih{
  • nama properti : nilai;
  • ....
  • }

Nilai adalah apa yang harus digunakan. Contoh:

  • sans-serif: Font sederhana tanpa dekorasi apa pun
  • cursive: Font yang terlihat seperti seseorang yang menulisnya, semacam itu
  • antiquewhite: Warna kuning kecoklatan muda
  • darkgreen: hijau tua
  • darkred: merah tua

Satukan semuanya.

Bagian pertama...

... mengatakan untuk membuat seluruh badan menggunakan font sans serif polos, dan membuat warna latar belakang menjadi cokelat. Jadi kita mendapatkan:

Semua teks dalam badan menggunakan font sans serif, kecuali h1.

Mengapa?

Karena kode CSS. Tepat setelah aturan body, ada aturan untuk h1.

h1 { warna: merah tua; keluarga font: cursif }

Benar! Aturan body tersebut memengaruhi seluruh halaman. Aturan h1 tersebut menggantikan aturan isi, hanya untuk elemen h1. 

Berikut HTML-nya lagi.​

  • <body>
  • <h1>Fakta sebenarnya tentang kucing</h1>
  • ...
  • </body>

Ada h1 di dalam body. Peramban web menerapkan aturan ke elemen luar terlebih dahulu, jadi semua yang ada di badan teks akan mendapatkan jenis huruf sans serif, termasuk h1. Kemudian peramban masuk ke dalam elemen luar, dan menggunakan aturan yang berlaku di sana. Jadi h1 akan mendapatkan jenis huruf kursif. 

Ini adalah bagian "berjenjang" dari "lembar gaya berjenjang" yang sedang bekerja.

Apakah urutan aturan dalam berkas CSS penting? Misalnya, jika aturan h1 tersebut ada sebelum aturan body lainnya, apakah itu akan mengubah banyak hal?

Beberapa HTML:

Teks untuk p berwarna hitam. Mengapa?

Anda agak membocorkannya di header bagian ini: default. Saya kira hitam adalah default, jika Anda tidak memberi warna?

Benar! Setiap properti CSS memiliki warna default. Anda dapat melihat warna teks default pada teks asli tanpa gaya.

Warna teks default adalah hitam, warna latar belakang default adalah putih, dan jenis huruf default adalah... seperti itu. Kelihatannya seperti Times Roman, atau semacamnya. 

Kecuali kita memberi tahu browser sebaliknya, propertinya akan tetap pada nilai default.

Benar juga, Ethan. Itu masuk akal, tetapi bukan begitu cara kerjanya. Berikut kode HTML-nya lagi:

  • <body>
  • ...
  • <h2>Cinta</h2>
  • <p>Kucingmu mencintaimu.</p>
  • </body>

H2 dan p dibungkus oleh atau disarangkan di body. Jadi, gaya yang diterapkan pada juga diterapkan pada h1 dan  p. Dalam istilah tag h2 dan p mewarisi properti body.

Namun, tag p muncul setelah Tag h2. Tag h2 ditutup, sebelum tag p dibuka. Tag p tidak terpengaruh oleh properti yang diterapkan pada h2. 

Basic styles

Ringkasan 

  • Lembar gaya memberi tahu peramban tampilan seperti apa yang harus diberikan pada HTML.
  • File-file stylesheet ditautkan dalam head.
  • Lembar gaya ada dalam bahasa CSS, bukan HTML.
  • Anda dapat mengatur font, warna, dan hal lainnya.
