Folder dan proyek 

Sekarang Anda memiliki folder di PC, mari kita buat agar mudah untuk bekerja dengan folder tersebut di VsCode. Mudah itu bagus. 

VsCode memiliki proyek. Proyek adalah sekumpulan file. Bagi kami, proyek adalah folder. Keduanya tidak sepenuhnya sama, tetapi cukup mirip untuk kursus kami. 

Ada dua cara untuk membuat proyek. Salah satunya adalah dengan membuat folder di Windows terlebih dahulu, lalu jalankan VsCode, dan perintahkan untuk menggunakan folder tersebut. Cara lainnya adalah dengan melakukan semuanya dalam satu langkah. 

Kita sudah membuat beberapa folder, jadi mari kita gunakan cara pertama.

Saya rasa lebih mudah membuat satu proyek VsCode yang berisi semua latihan. Dengan cara itu, lebih mudah menyalin kode.

Inilah hasil akhirnya:

Anda dapat melihat di mana saya meletakkan semuanya (1), dalam sebuah folder di dalam root web XAMPP (D:\xampp\htdocs). 

Saya membuat folder untuk setiap bagian kursus (2), dan di dalam folder tersebut untuk setiap latihan (3 dan 4). 

(Nama folder pada tangkapan layar itu mungkin tidak sama dengan nama pada tangkapan layar lainnya, tetapi Anda seharusnya mengerti maksudnya.)

Sebuah halaman 

Sebelumnya, Anda melihat beberapa HTML yang dapat Anda gunakan untuk templat halaman:

    1. <head>
    2. <title> Title</title>
    3. <meta charset="utf-8">
    4. <meta name="viewport" content="width=device-width, initial-scale=1,shrink-to-fit=no">
    5. </head>
    6. <body>
    7. <h1> Kucing berjalan di atas genting
    8. <p> Kucing Cantik sebelumnya di bawah, kemudian naik. Besoknya makan ikan</p>
    9. <p>Kemarin, hari ini dan besok kucing cantik berjalan.<p>
    10. </body>
    11. </html>

Mari kita sesuaikan, untuk membuat halaman seperti ini:

Buat file HTML baru 

Klik gambar pada akar proyek, dan buat file HTML baru.

Saya akan menamai berkas tersebut index.html. Mengapa saya memberi nama berkas tersebut index.html? Jika Anda tidak menyertakan nama berkas di URL, server web akan mencari berkas dengan nama default. Index.html adalah salah satu nama default.

Saya akan mengetikkan tanda seru ! , setelah itu tekan enter/klik. Ini akan mengaktifkan vscode untuk membuat file template html.
