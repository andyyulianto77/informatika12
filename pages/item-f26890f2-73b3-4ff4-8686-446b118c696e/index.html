Apa saja contoh dabase?

Anda akan membuat dua, satu di localhost(PC Anda), dan satu di InfinityFree. Ini adalah cara yang bagus untuk bekerja. Jalankan kode Anda di PC Anda, lalu buat duplikat DB di server Anda.

MySQL adalah salah satu sistem manajemen basis data relasional (RDBMS) yang paling populer dan banyak digunakan di dunia. Belajar MySQL dapat membuat aplikasi web dan bahasa pemrograman seperti PHP, Python, dan Java untuk membangun aplikasi web yang dinamis. Dengan MySQL, Anda dapat menyimpan dan mengelola data pengguna, produk, konten, dan informasi lainnya yang dibutuhkan oleh aplikasi Anda.

MySQL memungkinkan Anda untuk menyimpan dan mengelola data dalam jumlah besar. Anda dapat menggunakannya untuk melakukan berbagai analisis data, seperti membuat laporan, membuat visualisasi data, dan menemukan tren.

Tenaga terampil pada MySQL sangat dibutuhkan di pasar kerja. Banyak perusahaan membutuhkan profesional yang dapat merancang, membangun, dan mengelola database.

Pilih mana yang paling dipahami dari 3 tujuan di atas dan tulis alasannya

Host lokal

Di XAMPP CP (panel kontrol), jalankan Apache dan MySQL. Kemudian klik tombol DB admin.

phpMyAdmin akan dimulai. Ini adalah aplikasi web yang ditulis dalam PHP, jadi dimulai di browser.

Kini, MySQL memiliki sistem keamanannya sendiri, terpisah dari OS. Saat Anda mengakses DB MySQL, Anda melakukannya sebagai pengguna MySQL. Lebih mudah untuk membuat pengguna dan DB secara bersamaan.

Mengapa demikian?

Anda satu-satunya orang yang membuat DB di PC Anda. Jika Anda membuat DB bernama dbsiswa, hanya ada satu.

Akun server Anda berbeda. Untuk menekan biaya, Anda berbagi satu server Linux dengan orang lain. Apa yang terjadi jika beberapa dari Anda membuat DB siswa? Argh! Namanya bentrok.
