Mari dapatkan beberapa file pada subdomain latihan baru Anda.

Manajer File

Di cPanel, jalankan File Manager. Mirip seperti Explorer di Windows. Explorer menampilkan file-file di PC Anda. File Manager menampilkan file-file di server Anda.

FM memiliki bagian-bagian yang berbeda.

Ada folder di sebelah kiri. File dalam folder yang saat ini dipilih ada di sebelah kanan. Tindakan yang dapat Anda lakukan pada folder dan file, seperti menyalin dan mengunggah, ada di atas panel folder dan file.

Subdomain adalah folder

Di bagian folder, cari folder untuk subdomain latihan yang Anda buat. Nama folder saya adalah andy1.rf.gd(pada akun yang saya gunakan untuk pelajaran ini). Klik folder tersebut.

Folder subdomain Abaikan dua folder di dalam andy1.rf.gd cPanel membuatnya saat membuat subdomain Anda. Kami tidak akan menggunakannya.​

Semua berkas yang Anda masukkan ke dalam folder exercises.rf.gd tersebut akan muncul di situs web di https://exercises.yourdomain.com. Jadi, untuk menaruh berkas di web, cukup taruh berkas tersebut di folder di server Anda.

Serius? Anda tinggal menaruh file di sana, dan file itu sudah ada di web?

Itu benar.

Catatan

Dalam kasus ini, subdomain dan folder memiliki nama yang sama, exercises.yourdomain.com. Namun nama-nama tersebut tidak harus sama; cPanel hanya memberi nama secara default.

Setiap subdomain memiliki folder terkait.

Sama dengan domain utama Anda. Domain utama saya adalah andyux.my.id . Foldernya adalah public_html.

Semua berkas yang saya masukkan public_html akan ada di web di https://andyux.my.id .

Mengapa namanya public_html? Itulah nama yang dibuat Infinity. Folder untuk domain utama sering disebutpublic_html.

Tambahkan file ke subdomain Anda

Pilih folder untuk subdomain latihan Anda, dan klik tombol "+ File" untuk membuat file baru. Buat file baru.

Ketik nama berkas, dengan html ekstensi. Nama apa pun yang Anda suka, tetapi huruf kecil dan tanda hubung untuk spasi . Jadi bukan My File.Html , tetapi my-file.html. Membuat file

Edit berkas tersebut. Klik kanan pada berkas tersebut, lalu pilih Edit (bukan edit HTML). Edit

(Jika Anda mendapatkan pesan tentang penyandian teks dan UTF-8, Anda dapat mengabaikannya. Kami akan selalu menggunakan UTF-8 dalam kursus ini; faktanya, itulah satu-satunya yang saya gunakan, dalam proyek apa pun. Informasi lebih lanjut tentang apa yang dimaksud dengan UTF-8 akan dibahas nanti dalam kursus ini.)

Anda mengedit berkas langsung di server. Secara umum, ini bukan ide yang bagus. Lebih baik mengedit berkas di PC Anda sendiri, lalu mengunggahnya. Namun, untuk saat ini, mengedit di server akan lebih membantu, sehingga Anda dapat mempelajari cara kerjanya. Ketik apa pun yang Anda suka di berkas tersebut. Saya mengetik:

  • <h1>Kitty</h1>
  • <p>kucing bagus.</p>

Simpan berkas. Jangan gunakan Ctrl+S. Gunakan tombol "Simpan Perubahan" di kanan atas.

Ke peramban

Oke, Anda punya berkas di folder exercises.yourdomain.comserver Anda. Ingat bahwa berkas di folder tersebut harus ada di web di https://exercises.yourdomain.com.

Jadi, jika Anda membuat berkas kity.html di exercises.yourdomain.comserver Anda, berkas tersebut seharusnya tersedia di https://exercises.yourdomain.com/kity.html. Mari kita periksa.

Arahkan peramban Anda ke https://exercises.yourdomain.com/kity.html. Atau apa pun nama berkas Anda. Berikut ini yang saya lihat:

Intinya: untuk menaruh berkas di web, letakkan berkas tersebut dalam folder di server Anda.

Subfolder dan URL

Anda memerlukan berkas gambar untuk bagian ini. Anda dapat menggunakan berkas ini, jika Anda suka:​

Untuk menyimpan berkas gambar di PC Anda, klik kanan gambar tersebut, lalu pilih "Simpan gambar sebagai...", atau apa pun sebutan untuk tindakan ini di peramban Anda.

Gunakan gambar lain, jika Anda mau. Pastikan ekstensinya adalah jpgatau png.

Buat subfolder di exercises.yourdomain.com, yang disebut images. Ada tombol "+ Folder" di FM.

Buka images(klik dua kali), dan di dalamnya, buat folder lain bernama cute-kuncing. Atau cute-turtles, atau apa pun yang Anda suka. Itu adalah subfolder dari images, yang merupakan subfolder dari folder subdomain Anda. Jadi itu adalah subsubfolder. Namun itu menjadi rumit, jadi kita akan menyebutnya folder saja.

Buka folder yang baru Anda buat.

Sekarang unggah berkas gambar Anda. File Manager memiliki tombol yang akan mengunggah berkas ke folder saat ini.

Petunjuk

Jika Anda tidak melihat berkas atau folder, gunakan item menu Muat Ulang di FM:

Jadi sekarang, Anda seharusnya memiliki berkas gambar di subsubfolder yang Anda buat.

Periksa gambar di browser

Jalur ke gambar di server Anda adalah exercises.yourdomain.com/images/cute-dogs/rosie1.jpg. Atau apa pun nama yang Anda berikan. 

Sekarang, folder server exercises.yourdomain.com dipetakan ke URL https://exercises.yourdomain.com. Jadi, peramban Anda seharusnya dapat menemukan berkas gambar di https://exercises.yourdomain.com/images/cute-kit/kity.jpg. 

Ketik URL Anda, dan lihat apakah peramban Anda bisa mendapatkan gambar dari situs Anda. Oren! Dia manis sekali!

Dia manis sekali!

Latihan File dalam subfolder

Temukan gambar hewan yang Anda sukai. Seekor llama, kucing anjing, gurita, apa saja. Taruh di folder pada subdomain latihan Anda. Folder di dalam folder juga boleh, sedalam apa pun yang Anda inginkan.

Kirimkan URL gambar.

Ringkasan 

  • Tiap subdomain dipetakan ke folder di server.
  • Taruhlah suatu berkas dalam folder itu, dan berkas itu akan ada di web.​
