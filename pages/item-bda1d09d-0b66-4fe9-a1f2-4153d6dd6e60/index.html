Baris pertama...

  • <!doctype html>

... memberi tahu browser bahwa halaman tersebut menggunakan HTML, bukan bahasa lain, seperti XML. 

Berikutnya, seluruh halaman ada dalam sebuah html tag.​

  • <!doctype html>
  • <html lang="en">
  • sesuatu hal
  • </html>

Tag html memiliki bagian pembuka dan bagian penutup:

  • <!doctype html>
  • <html lang="en">
  • hal-hal
  • </html>

Bagian penutup memiliki tanda / di depan. Sebagian besar tag memiliki bagian pembuka dan penutup, meskipun tidak semuanya. Ini dia lagi:

  • <!doctype html>
  • <html lang="en">
  • hal-hal
  • </html>

Tag membungkus tag-tag di dalamnya. Anda akan melihat kata itu - membungkus - banyak. Tag di dalam tag merupakan hal penting dalam HTML. Kata lain untuk itu adalah html bersarang. Hal-hal bersarang di dalam tag.html

Satu hal lagi tentang html tag.

  • <!doctype html>
  • <html lang="en">
  • hal-hal
  • </html>

lang adalah atribut , atau properti dari tag. Yang ini memiliki nilai en, yang memberi tahu browser bahwa halaman tersebut berbahasa Inggris. Anda akan melihat properti lainnya saat kita melakukannya.
