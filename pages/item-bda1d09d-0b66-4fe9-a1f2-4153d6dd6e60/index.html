Set karakter

Berikut bagian atas halaman lagi:

  1. <!doctype html>
  2. <html lang="en">
  3. <head>
  4. <title> Title</title>
  5. <meta charset="utf-8">
  6. <meta name="viewport" content="width=device-width, initial-scale=1,shrink-to-fit=no">
  7. </head>

Baris 5 adalah

Set karakter adalah set simbol yang digunakan komputer untuk halaman web, dokumen Word, PDF, atau apa pun. Ada banyak set karakter. UTF-8 adalah salah satunya, dan berisi semua karakter yang kita gunakan dalam bahasa Inggris, ditambah ribuan karakter yang digunakan dalam bahasa lain. 

  • <meta charset="utf-8">

Selalu gunakan UTF-8 dalam kursus ini.  Hampir semua halaman web melakukannya.

Jendela pandang/Viewport

  1. <!doctype html>
  2. <html lang="en">
  3. <head>
  4. <title> Title</title>
  5. <meta charset="utf-8">
  6. <meta name="viewport" content="width=device-width, initial-scale=1,shrink-to-fit=no">
  7. </head>

Baris 6 adalah: 

  • <meta name="viewport" content="width=device-width, initial-scale=1,shrink-to-fit=no">

Tag viewport ini berlaku pada perangkat seluler. Nantinya dalam kursus ini, Anda akan membuat halaman yang dapat disesuaikan dengan ukuran layar yang berbeda, sehingga dapat digunakan pada laptop, tablet, ponsel, dan lain-lain. Tag viewportini membantu dalam hal itu.

Ringkasan

Setiap halaman yang Anda buat setidaknya harus memiliki tag berikut:

  1. <head>
  2. <title> Title</title>
  3. <meta charset="utf-8">
  4. <meta name="viewport" content="width=device-width, initial-scale=1,shrink-to-fit=no">
  5. </head>
  6. <body>
  7. <p> Kucing bagus</p>
  8. </body>
  9. </html>
