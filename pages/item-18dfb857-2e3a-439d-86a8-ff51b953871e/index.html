DAPATKAN itu

Cara paling sederhana untuk mengirim data dari browser ke server adalah melalui URL.

Berikut URL ke halaman pada web latihan:

  • http://andy1.rf.gd/php/db/1-n/find-household/carianjing.php?household_name=mathieson
URL-nya memiliki:
  • Protokol,https
  • Pelayannya,andy1.rf.gd
  • Jalur halaman,/php/db/1-n/find-household/carianjing.php
  • Dan beberapa hal aneh di akhir,?household_name=mathieson

Hal-hal di bagian akhir adalah string kueri . Itu ?menandakan awal string. Ada parameter dan nilai. Jadi parameter idmemiliki nilai mathieson.

Anda dapat meneruskan lebih dari satu parameter, seperti ini:

  • sesuatu?nama=Oren&kelucuan=12

Tanda ampersand (&) memisahkan argumen. Anda dapat memasukkan banyak argumen.

Siapa yang paling imut?

Berikut kode dari terakhir kali:

    1. $cutestCat ​​= 'Oren';
    2. $peringkat = 12;
    3. ?><!DOCTYPE html>
    4. <html lang="id">

Mari kita ubah baris 2 dan 3, untuk mendapatkan data dari URL, seperti ini:

  • $cutestCat ​​= $_GET['nama'];
  • $rating = $_GET['kelucuan'];

Jadi program akan mengharapkan dua parameter di URL: name, dan cuteness.

Coba URL ini:

http://andy1.rf.gd/html/enter-php/cantik.php?nama=Persia&kelucuan=12

Di peramban Anda, ubah nama dan peringkat di URL, dan lihat apa yang terjadi.

Cobalah untuk menghilangkan data, yaitu ?dan semua yang setelahnya.

Perhitungan

Anda dapat melakukan perhitungan, seperti di VBA. Berikut contoh lainnya:

  1. <?php
  2. $a = $_GET['a'];
  3. $b = $_GET['b'];
  4. ?><!DOCTYPE html>
  5. <html lang="id">
  6. <head>
  7. <meta charset="UTF-8">
  8. <title>Perhitungan</title>
  9. <meta name="viewport" content="width=device-width, initial-scale=1, shrink-to-fit=no"&gt; </head>
  10. <body>
  11. <h1>Perhitungan</h1>
  12. <?php
  13. $aKaliB = $a * $b;
  14. print "<p>$a dikali $b adalah $aKaliB.</p>";
  15. $aDibagiDenganB = $a / $b;
  16. print "<p>$a dibagi $b adalah $aDibagiDenganB</p>";
  17. ?>
  18. </body>
  19. </html>

Lihat baris 14:

  • print "<p>$a dikali $b adalah $aKaliB.</p>";

Jika Anda menggunakan tanda kutip ganda (") di sekitar string, Anda dapat meletakkan nama variabel di dalam string tersebut, dan PHP akan mengganti nilainya. Hal ini tidak berlaku untuk tanda kutip tunggal. Jadi...

  • print '<p>$a dikali $b adalah $aKaliB.</p>';

... akan menghasilkan output persis seperti ini <p>$a times $b is $aTimesB.</p>, dengan tanda dolar dan nama variabel.

Terkadang Anda perlu menambahkan tanda kurung kurawal, jika PHP tidak dapat memahami maksud Anda. Misalnya, Anda dapat menggunakan...

  • print cetak '<p>{$a} dikali {$b} adalah {$aTimesB}.</p>';

Tanda kurung memberi tahu PHP bahwa hal di antara keduanya harus diganti dengan suatu nilai. Tanda kurung itu sendiri tidak ditampilkan.

Kode yang kita lihat sejauh ini tidak memerlukan kurung kurawal, tetapi Anda akan menemukannya nanti dalam kursus ini.

Perhitungan bisa sepanjang yang Anda inginkan

PHP dapat melakukan perhitungan yang rumit, seperti VBA. Misalnya:

  • $hasil = ($x + $y) * 4 - 27 / (90 + $z / 2,33);

() berfungsi sama seperti di VBA. Urutan operasinya sesuai dengan yang Anda harapkan.

Apa yang akan dihasilkan kode ini?

  • $hasil = 2 + 3 * 4;
  • cetak $result;
Jika Anda masuk sebagai siswa, pelajaran akan berhenti di sini, dan Anda akan diminta untuk mengetik jawaban. Jika Anda ingin mencobanya, mintalah akun di situs ini.
