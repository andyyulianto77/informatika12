Isi dari database adalah kumpulan tabel-tabel

Pada akhir pelajaran, siswa seharusnya dapat ...

  1. Membedakan field dan record.
  2. Menulis dan menjalankan perintah SQL untuk membuat tabel dengan struktur yang benar.
  3. Mengganti nama tabel dengan perintah SQL..

Mari kita membuat tabel tentang localhost.

NISadalah INTkependekan dari integer, yaitu bilangan bulat. Ini adalah kunci utama tabel. Setiap siswa akan memiliki id yang berbeda, jadi meskipun dua siswa memiliki nama yang sama, program kami dapat menggunakan id tersebut untuk membedakannya.

NISakan bertambah secara otomatis. Saat kita menambahkan data baru ke dalam tabel, DBMS akan secara otomatis membuat nilai id baru untuk data tersebut. Satu hal yang tidak perlu kita pikirkan lagi.

NIStidak boleh null. (Kita sudah membahas null sebelumnya.) "Tidak null" berarti kolom tersebut wajib diisi. Setiap rekaman siswa harus memiliki id.

nameadalah VARCHAR, yang berarti dapat menampung sejumlah karakter yang bervariasi. Hingga 70, untuk bidang ini. Tidak boleh null, artinya, ini adalah bidang yang wajib diisi untuk setiap rekaman.

weightadalah FLOAT. Seperti Singledalam VBA, sebuah angka, seperti 2.1. Itu bisa bernilai null, artinya tidak diperlukan. Jika bernilai null, kita tidak tahu berat siswa.

commentsadalah TEXT. Apa pun yang dapat Anda ketik, hingga 65.535 karakter. Bisa berupa null, jadi tidak diperlukan.

Ekspor DB ke server

Oke, kita punya database di localhost. Mari ekspor apa yang kita punya ke server.

Jalankan phpMyAdmin di localhost. Setelah siswamemilih DB, pilih tab Ekspor.

Biarkan semua pengaturan sebagaimana adanya, dan klik tombol Go (tombol Go, tombol Go, Go!). phpMyAdmin akan membuat file dengan perintah yang dibutuhkan untuk membuat ulang tabel, dan mengirimkannya ke PC Anda. Simpan file tersebut di suatu tempat.

(Jika Anda mau, buka berkasnya di Notepad dan periksa.)


Sekarang jalankan phpMyAdmin di server Anda (dari cPanel), dan pilih DB siswa. Ingat, DB tersebut akan diberi nama siswa, atau apa pun nama pengguna Linux Anda.

Klik tab Impor, klik tombol Pilih file, dan pilih file yang baru Anda simpan.

Sekarang klik Go. phpMyAdmin akan membaca berkas tersebut, dan menjalankan pernyataan SQL di dalamnya.

Lihat pada tab Struktur, dan tabel siswa akan tercantum. Lihat struktur tabel, dan Anda akan melihat kolom yang Anda buat di localhost.
